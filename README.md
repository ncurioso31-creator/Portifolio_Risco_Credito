Principais Insights de Negócio

Renda: O Poder Aquisitivo como Critério Principal de Aprovação.
Ao analisar a relação entre o Score de Crédito (eixo Y) e a Renda Anual(eixo x), fica claro que o critério de maior peso utilizado para aprovação é o poder aquisitivo, os 20.000 clientes dessa base, aqueles com uma renda mais elevada apresentam probabilidade muito maior de obter crédito, independentemente da sua pontuação de score.

° O insight: A capacidade de pagamento(Renda Anual) dita a regra. Mesmo com score considerado bom, é o poder aquisitivo que acaba sendo o fator decisivo para o banco.

° Fatos: Como mostra no gráfico abaixo, há uma concentração massiva de empréstimos negados (pontos azuis) em faixas de renda inferiores a $60.000,ainda que possuam um score médio aceitável (entre 500 e 600). Por outro lado, a maioria esmagadora dos empréstimos aprovados(pontos vermelhos) concentra-se acima de 80.000. Fica evidente que, conforme a renda do cliente aumenta, o volume de aprovações cresce na mesma proporção.
